开始使用

基于Vue3

享受 Vue3 带来的性能提升，更好的开发体验。基于 Composition API / TSX 内置了诸多特性，外部调用只需简单配置即可呈现丰富的功能。

适合业务

对于 Pro 组件，其本质上是基于 AntD 组件的二次封装。在设计上，Pro 组件并没有裁剪掉 AntD 组件的任何能力，而是一种适合业务的能力加强，简单的结构化数据配置，即可展开为复杂的表单/表格，同时支持自定义渲染，做到张弛有度。

拥抱ESM & TS

提供 CJS/UMD 输出的同时，也提供了 ESM 输出，配合按需加载实现最小化引入，提供完整的 Typescript 支持。配合 unplugin-vue-components 可以做到自动按需引入，自动生成 components.d.ts，结合 volar 提供完备的类型支持。
